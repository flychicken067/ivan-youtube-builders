# Digest framing rules

Philosophy: **Follow builders, not KOLs.**

A KOL talks about what others built. A builder shows you what they made.
This digest only covers the latter.

Digest structure:
1. One-line header with date and number of new videos
2. For each builder with a new video: name → what they built → why it matters → URL
3. If no new videos in the last 48h for a builder, skip them entirely
4. Closing line: "Next digest at [scheduled time]." or "Type /yt for another digest."

Tone: direct, no hype. The work speaks. You translate it.
